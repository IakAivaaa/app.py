# 🚀 EnergyWise AI: Neural ESG Dashboard

Το **EnergyWise AI** είναι μια προηγμένη εφαρμογή ανάλυσης ενέργειας (Proof of Concept) που συνδυάζει **Machine Learning**, **ESG Analytics** και **Interactive Data Visualization**. Στόχος της είναι να παρέχει στους καταναλωτές πλήρη διαφάνεια για την κατανάλωση ρεύματος σε επίπεδο συσκευής.

## 🌟 Βασικά Χαρακτηριστικά
- **AI Load Disaggregation:** Επιμερισμός της συνολικής κατανάλωσης σε μεμονωμένες συσκευές (NILM Logic).
- **ESG & Sustainability Tracking:** Υπολογισμός αποτυπώματος CO2 και αντιστοίχιση σε περιβαλλοντικό αντίκτυπο (αριθμός δέντρων).
- **EU Benchmarking:** Σύγκριση της οικιακής κατανάλωσης με τον ευρωπαϊκό μέσο όρο.
- **Interactive AI Training:** Δυνατότητα εκπαίδευσης του μοντέλου από τον χρήστη για βελτίωση της ακρίβειας αναγνώρισης συσκευών.
- **Professional Enterprise Reporting:** Εξαγωγή ολοκληρωμένου report σε Excel με αυτοματοποιημένα γραφήματα και αναλύσεις.

## 🛠️ Τεχνολογίες
- **Python 3.x**
- **Streamlit** (Web Interface)
- **Pandas & NumPy** (Data Processing)
- **Plotly & XlsxWriter** (Visualizations & Reporting)
